module的两种导入和导出 export default 导出和对应的import导入 exprot 导出和对应的import导入

export default 导出和对应的import导入 1.认识导入和导出 导出的东西可以被导入(import)并访问到 一个模块没有导出,也可以将其导入 被导入的代码都会执行一遍,也仅会执行一遍 import './module.js';多次导入都执行一边 2.基本用法 可以随便起名 import age from './module.js'; 这里就能调用文件中的age变量了 //一个模块只能有一个export default //export default age; //export default 20; //export default fn(箭头函数); //export default class{}; //export default function (){}; //export default {};

export导出和对应的import导入 基本用法 多个导出 导出导入时起别名 整体导入 同时导入 1.基本用法 export 声明或者语句 可以和export default 一起使用 例如 export const age=18; export {age}；也可以 //不能随意命名 import {age} from './module.js'; 2.多个导出 可以写一个导一个 import {fn,age,className} from './module.js'; 3.导出导入时起别名 export {fn as fnc,className,age}; import {fnc,age,className as person} from './module.js'; 4.整体导入 import * as obj from './module.js'; 5.同时导入 一定是export default的在前,不然会报错 加个逗号合并 import ages,{fnc,age,className as person} from './module.js';
